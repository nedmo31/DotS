Transactions : 
    POST    /trade     send JSON : {uid, isBuy, tid, amount}    e.g. {1, true, 2, 4}  user 1 buys 4 shares of team 2

    POST    /login    send JSON : {username, password}         returns JSON {uid}


Data Pull : 
    GET     /users          returns JSON {[{uid, username, networth, money, []}, ...]} returns of a list of users with their name and net worth

    GET     /users/:uid     returns JSON {uid, username, networth, money, ownerships:[{uid, tid, count, name}, ...]} returns a user and their list of stocks

    GET     /teams          returns JSON { [ {tid, name, price, wins, losses, pointsfor, pointsagainst}, ...] }
